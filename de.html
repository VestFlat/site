Kein Hosting nötig
Ihre Website ist in der Cloud gespeichert. Kurze Ladezeiten und die Sicherheit vor DDoS-Attacken sind gewährleistet.
Die kurzen Ladezeiten werden mittels Content Delivery Network (CDN) erreicht. Ein SSL-Zertifikat (benötigt zur Konfiguration von HTTPS) kann in Ihrem persönlichen Konto aktiviert werden.
Export and API
Eine Site kann problemlos auf einen anderen Server exportiert oder über ein Web-API in eine fertige Website integriert werden.
